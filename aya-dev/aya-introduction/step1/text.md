You need to install Rust along with all the required tools using rustup.

<br>

### Solution
Install Rustup:

```plain
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```{{exec}}

To ensure the correct binary path is set, run the following command: 

```plain
. "$HOME/.cargo/env"
```{{exec}}
